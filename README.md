📄 Student Management System (Short)


📌 Project Description
Student Management System is a web-based application developed using Java, Servlets, JDBC, HTML, and CSS. It is used to manage student data by performing CRUD operations like add, view, update, and delete student records. The system connects to a database to store and retrieve data.


⭐ Features
Add student
View student
Update student
Delete student


💻 Technology
Java (Core Java)
Servlet
JDBC
HTML, CSS
MySQL
Apache Tomcat


🗄️ Database Configuration
Database Name: studentdb
Table: students (id, name, email, course)
JDBC URL:

jdbc:mysql://localhost:3306/studentdb


▶️ How to Run
Start MySQL and create database
Import project in IDE (Eclipse)
Configure Tomcat server
Run project on server
Open browser:

http://localhost:8080/StudentCRUD/
